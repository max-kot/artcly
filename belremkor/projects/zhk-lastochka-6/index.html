О проекте

Проектом предусматривается размещение проектируемого 60-ти квартирного жилого дома, разработанного ООО «БелРемКор».
Проектируемое здание размещается в южной части г. Бреста.

С северо-запада строительная площадка ограничена многоэтажной жилой застройкой, с восточной стороны – индивидуальной жилой застройкой, с юга – территория ограниченна ул. Подгородской.

На благоустраиваемом участке проектируемого объекта проектом предусмотрено размещение:
— парковки для индивидуального автотранспорта на 10 машино-мест с въездом с ул. Подгородская, 50 машино-мест предусмотрено на подземном паркинге, в т.ч. места для физически ослабленных лиц. Конструкция дорожной одежды (КДО) на парковке запроектирована из мелкоштучной плитки толщиной 0,08м с цветографическим выделением по гравийному основанию с учетом постоянной транспортной нагрузки ;
— детской площадки для различных возрастных групп, а также площадки Workout. КДО площадок представлено покрытием из резиновой плитки ;
— мусорной камеры, встроенной в жилое здание. Подъезд к мусорной камере предусмотрен с существующего проезда со стороны многоэтажной жилой застройки с КДО по типу 1;
— пешеходных путей с КДО с учётом и без учёта транспортной нагрузки;
— 3D ограждения территории в границах землевладения;
— усиленной конструкции газона для возможности проезда спецтранспорта ;
— велопарковок на первом этаже жилого дома.
